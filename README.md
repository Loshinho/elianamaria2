# Eliana Maria - Ateliê de Beleza

Site institucional premium para salão de beleza especializado em Dia da Noiva.

## Como Executar Localmente

**Pré-requisitos:** Node.js instalado.

1. Instale as dependências:
   `npm install`

2. Inicie o servidor de desenvolvimento:
   `npm run dev`
